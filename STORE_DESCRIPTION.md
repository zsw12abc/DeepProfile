# DeepProfile - AI驱动的社交媒体用户画像分析工具

## 应用商店主描述

DeepProfile 是一款先进的 Chrome 浏览器扩展，利用 AI 技术（支持 OpenAI、Gemini、Ollama、通义千问等）分析社交媒体用户的近期动态，生成多维度的性格与领域偏好画像，助您更好地理解和理性交流。

### 主要功能
**🤖 AI 驱动分析**
- 支持多种 AI 服务提供商：OpenAI (GPT-4/GPT-3.5)、Google Gemini、Ollama (本地模型)、通义千问、DeepSeek 等
- 智能分析用户的政治倾向、经济观点、社会态度、科技观点、文化偏好等多维度特征

**🌐 多平台支持**
- 目前支持知乎和 Reddit 平台
- 自动在用户链接旁添加分析按钮，一键查看用户画像
- 智能识别用户内容，提供精准分析

**⚡ 智能分析模式**
- **极速模式**：快速概览，最低 Token 消耗
- **平衡模式**：兼顾质量与速度的默认选择
- **深度模式**：深入分析，识别讽刺、隐喻等复杂内容

**💾 智能缓存系统**
- 自动缓存分析结果，避免重复调用 API
- 支持历史记录管理，随时查看过往分析
- 支持导出分析结果为图片或 Markdown 格式

**🎨 个性化主题**
- 支持多种预设主题（知乎白/黑主题、Reddit白/黑主题等）
- 可自定义颜色方案、字体、尺寸等外观设置
- 实时主题切换，无需刷新页面

**🔒 隐私安全**
- 所有 API 密钥仅存储在本地浏览器，绝不上传服务器
- 不收集任何用户数据或浏览记录
- 符合 Chrome 扩展隐私政策要求

### 技术特色
- **多维度分析**：基于 8 大类 40+ 个细分维度进行深度分析
- **实时设置保存**：配置更改立即生效，无需手动保存
- **多语言支持**：支持中文和英文界面切换
- **上下文感知**：根据当前浏览话题自动调整分析重点
- **证据引用**：提供可点击的原文引用链接

### 使用场景
- 理性交流：了解对方观点背景，促进理性讨论
- 内容审核：辅助判断内容倾向性
- 学术研究：社交媒体用户行为分析
- 社交辅助：更好地理解社区成员

### 隐私承诺
我们承诺您的数据安全：
- 所有配置数据仅本地存储
- 不收集任何用户浏览数据
- 不使用任何追踪或分析工具
- API 密钥仅在本地使用

### 如何使用
1. 安装扩展后，在设置页面配置您喜欢的 AI 服务提供商和 API 密钥
2. 访问知乎或 Reddit 页面，您会在用户链接旁看到 🔍 分析按钮
3. 点击按钮即可查看该用户的多维度画像分析
4. 查看历史记录、导出结果或自定义主题设置
